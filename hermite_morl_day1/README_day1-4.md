# Día 1-4 — Hermite-Gauss base

Ejecutar desde la raíz del proyecto:

```bash
pip install -r requirements.txt
python scripts/run_day1.py --config configs/default.yaml
```

Si se agregan imágenes propias, colocarlas en `data/raw/`. Si la carpeta está vacía, se usa un dataset pequeño de `skimage.data` para pruebas rápidas.
